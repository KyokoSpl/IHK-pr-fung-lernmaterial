# Vertiefung: 4.3.02 – Gefaehrdungen und Beanspruchungen wahrnehmen und einschaetzen

## Gefaehrdungsbeurteilung – Ablauf im Detail

```
  Schritt 1: Arbeitsbereiche und Taetigkeiten erfassen
         |
         v
  Schritt 2: Gefaehrdungen ermitteln
         |
         v
  Schritt 3: Gefaehrdungen bewerten (Risiko = Eintrittswahrscheinlichkeit x Schadensausmass)
         |
         v
  Schritt 4: Massnahmen festlegen (nach TOP-Prinzip)
         |
         v
  Schritt 5: Massnahmen durchfuehren
         |
         v
  Schritt 6: Wirksamkeit pruefen
         |
         v
  Schritt 7: Dokumentation und Fortschreibung
```

## Belastungs-Beanspruchungs-Modell

```
  +------------------+         +-------------------+
  |   BELASTUNG      |         |   BEANSPRUCHUNG   |
  | (objektiv,       | ------> | (subjektiv,       |
  |  messbar)        |         |  individuell)     |
  +------------------+         +-------------------+
  | Bildschirmarbeit |         | Person A: keine   |
  | 8 Stunden        |         |   Beschwerden     |
  | Raumtemp. 26°C   |         | Person B: Kopf-   |
  | Laerm 50 dB(A)   |         |   schmerzen,      |
  +------------------+         |   Augenreizung    |
                               +-------------------+

  Einflussfaktoren: Alter, Konstitution, Vorerkrankungen,
  Erfahrung, psychische Verfassung
```

## Vergleich: Physische vs. Psychische Belastungen am IT-Arbeitsplatz

| Aspekt | Physische Belastungen | Psychische Belastungen |
|--------|----------------------|----------------------|
| Art | Koerperlich messbar | Subjektiv empfunden |
| Beispiele | Fehlhaltung, Laerm, Hitze | Zeitdruck, Mobbing, Monotonie |
| Erkennung | Oft sichtbar/messbar | Schwer zu erkennen |
| Folgen (kurzfristig) | Schmerzen, Verletzungen | Stress, Konzentrationsverlust |
| Folgen (langfristig) | Muskel-Skelett-Erkrankungen | Burnout, Depression |
| Praevention | Ergonomie, Pausen, PSA | Arbeitsorganisation, Gespraeche |
| Rechtsgrundlage | ArbStaettV, DGUV | §5 ArbSchG (seit 2013 explizit) |
| Messung | Lux, dB(A), °C | Mitarbeiterbefragung, Interviews |

## Ersthelfer – Vertiefung

### Berechnung der Ersthelferanzahl

**Beispiel:** IT-Unternehmen mit 80 Buero-Beschaeftigten:
- Verwaltungsbereich → 5 % Regelung
- 80 × 0,05 = 4 Ersthelfer (Minimum)

### Ersthelfer-Infrastruktur

| Element | Anforderung |
|---------|-------------|
| Verbandkasten | Mindestens 1 pro Bereich (DIN 13157 klein / DIN 13169 gross) |
| Standort | Leicht zugaenglich, gekennzeichnet (weisses Kreuz auf gruenem Grund) |
| Ersthelfer-Aushang | Name, Telefon, Standort des Verbandkastens |
| Unfallmeldung | Verbandbuch fuehren (Datenschutz beachten!) |

## Typische Pruefungsaspekte

- Gefaehrdungsbeurteilung: Schritte benennen und auf ein Szenario anwenden
- Belastungs-Beanspruchungs-Modell erklaeren
- Ersthelferquote berechnen (5 % Verwaltung, 10 % sonstige)
- Physische Gefaehrdungskategorien mit IT-Beispielen zuordnen
- Fuersorgepflicht des AG vs. Mitwirkungspflicht des AN
- Psychische Belastungen am IT-Arbeitsplatz erkennen
- TOP-Prinzip auf konkrete Situationen anwenden

## Haeufige Fehler

| Fehler | Richtigstellung |
|--------|----------------|
| Gefaehrdungsbeurteilung nur einmalig noetig | Muss regelmaessig aktualisiert werden (bei Aenderungen, nach Unfaellen) |
| Psychische Belastungen sind kein Arbeitsschutzthema | Seit 2013 explizit in §5 ArbSchG aufgenommen |
| Ersthelfer muessen Aerzte sein | Ersthelfer sind Laien mit 9 UE Erste-Hilfe-Kurs |
| ESD betrifft nur Elektronikwerkstaetten | Auch im Buero relevant (z.B. beim Oeffnen von PCs, RAM-Einbau) |
| Beinahe-Unfaelle muss man nicht melden | Sollten gemeldet werden, um Praevention zu verbessern |
| Nur der AG ist fuer Sicherheit verantwortlich | AN hat Mitwirkungspflicht (§15, §16 ArbSchG) |
| Fuersorgepflicht endet am Werkstor | Gilt auch fuer Homeoffice und Dienstfahrten |
