# Vertiefung: 4.3.07 – Verhaltensweisen im Brandfall sowie vorbeugender Brandschutz

## Brandklassen und Loeschmittel – Zuordnungstabelle

```
  Brandklasse:     A          B          C          D          F
  Brennstoff:    Fest      Fluessig     Gas      Metall   Speisefett
                  |           |          |          |          |
  Wasser:        JA         NEIN       NEIN      NEIN!!     NEIN!!
  Schaum:        JA          JA        NEIN       NEIN       NEIN
  ABC-Pulver:    JA          JA         JA        NEIN       NEIN
  BC-Pulver:    NEIN         JA         JA        NEIN       NEIN
  CO2:          NEIN         JA        NEIN       NEIN       NEIN
  Metallpulver: NEIN        NEIN       NEIN        JA        NEIN
  Fettloescher: NEIN        NEIN       NEIN       NEIN        JA
```

**MERKSATZ fuer die Pruefung:**
- **Brandklasse D (Metalle):** NIEMALS Wasser! → Knallgasexplosion!
- **Brandklasse F (Fett):** NIEMALS Wasser! → Fettexplosion!
- **CO2 im Serverraum:** Beste Wahl, da rueckstandsfrei, aber Erstickungsgefahr in geschlossenen Raeumen

## Feuerloescher – Richtiger Einsatz

```
  BEDIENUNG EINES FEUERLOESCHERS:

  1. Feuerloescher aus Halterung nehmen
  2. Sicherungsstift herausziehen
  3. Pruefung: Windrichtung beachten (Wind im Ruecken)
  4. Schlauch auf Brandherd richten
  5. Hebel druecken → Loesch-Stoss abgeben
  6. Von vorne nach hinten loeschen
  7. Von unten nach oben loeschen
  8. Flaechenbrand: Von vorne loeschen

  REGELN:
  +-- Mehrere Loescher gleichzeitig (nicht nacheinander!)
  +-- Tropf- und Fliessbrand: Von oben nach unten
  +-- Wandbrand: Von unten nach oben
  +-- Wiederentzuendung beobachten
  +-- Benutzte Loescher sofort nachfuellen lassen
```

## Vorbeugender Brandschutz im Serverraum

### Brandschutzkonzept fuer IT-Raeume

| Massnahme | Kategorie | Beschreibung |
|-----------|----------|-------------|
| Feuerfeste Waende/Decken (F90) | Baulich | Feuerwidersstand mindestens 90 Minuten |
| Brandschutzklappe in Lueftung | Baulich | Verhindert Brandausbreitung ueber Kanaele |
| Brandmeldeanlage mit Rauchmelder | Technisch | Fruehzeitige Branderkennung |
| Gasloeschanlage (Argon/Stickstoff/Novec) | Technisch | Rueckstandsfreie Loeschung |
| CO2-Feuerloescher | Technisch | Fuer Entstehungsbraende, rueckstandsfrei |
| Temperaturueberwachung | Technisch | Monitoring mit Alarmierung |
| Rauchverbot | Organisatorisch | Klare Beschilderung |
| Brandschutzunterweisung | Organisatorisch | Jaehrlich fuer alle mit Serverraum-Zugang |
| Keine brennbaren Materialien | Organisatorisch | Kein Papier, Kartons im Serverraum |
| Zutrittskontrolle | Organisatorisch | Nur autorisiertes Personal |

### Gasloeschanlage vs. CO2-Feuerloescher

| Aspekt | Gasloeschanlage | CO2-Handloescher |
|--------|----------------|-----------------|
| Einsatz | Grosse Serverraeume, Rechenzentren | Kleine IT-Raeume, einzelne Geraete |
| Ausloesung | Automatisch (oder manuell) | Manuell |
| Loeschmittel | Inertgas (Argon, Stickstoff) oder Novec | CO2 (Kohlenstoffdioxid) |
| Personen im Raum | Vorwarnzeit, Raum muss geraeumt werden | Erstickungsgefahr bei Einsatz in kleinen Raeumen |
| Rueckstaende | Keine | Keine |
| Kosten | Hoch (Installation + Wartung) | Gering |

## Verhalten im Brandfall – Reihenfolge

```
  1. RUHE BEWAHREN
       |
       v
  2. BRAND MELDEN
     +-- Feuermelder betaetigen
     +-- Notruf 112
       |
       v
  3. IN SICHERHEIT BRINGEN
     +-- Gefaehrdete Personen warnen
     +-- Gebaeude ueber Fluchtwege verlassen
     +-- Tueren schliessen (Brandausbreitung verlangsamen)
     +-- KEIN Aufzug!
       |
       v
  4. LOESCHVERSUCH (nur bei Entstehungsbrand!)
     +-- Nur wenn eigene Sicherheit gewaehrleistet
     +-- Richtigen Loescher waehlen (Brandklasse!)
     +-- Bei Misserfolg: Sofort Raum verlassen
       |
       v
  5. SAMMELPLATZ
     +-- Vollzaehligkeitskontrolle
     +-- Feuerwehr einweisen
     +-- Fehlende Personen melden
```

**WICHTIG:** Reihenfolge: **MELDEN → RETTEN → LOESCHEN** (nicht umgekehrt!)

## Typische Pruefungsaspekte

- Brandklassen mit passenden Loeschmitteln zuordnen
- Warum ist CO2 der ideale Loescher fuer IT-Raeume?
- Brandschutzordnung Teil A/B/C unterscheiden
- Reihenfolge im Brandfall (Melden → Retten → Loeschen)
- Warum KEIN Wasser bei Brandklasse D und F?
- Feuerloescher-Bedienung (Stift ziehen, Wind im Ruecken, von unten)
- Vorbeugender Brandschutz im Serverraum nennen

## Haeufige Fehler

| Fehler | Richtigstellung |
|--------|----------------|
| Erst loeschen, dann melden | Immer ZUERST melden (Feuermelder + 112), dann loeschen |
| Wasser loescht jeden Brand | Wasser ist nur fuer Brandklasse A. Bei D und F: Explosionsgefahr! |
| ABC-Pulver ist ideal fuer Serverraeume | ABC-Pulver hinterlaesst Rueckstaende → zerstoert IT-Hardware |
| CO2-Loescher fuer alle Brandklassen | CO2 wirkt nur bei Brandklasse B (und begrenzt bei A) |
| Tueren im Brandfall oeffnen/offenstehen lassen | Tueren SCHLIESSEN → verhindert Rauch- und Brandausbreitung |
| Aufzug bei Brand benutzen | Aufzuege NIEMALS im Brandfall benutzen (Rauchfalle, Stromausfall) |
| Brandschutzordnung Teil A nur fuer Mitarbeiter | Teil A richtet sich an ALLE Personen im Gebaeude (auch Besucher) |
| Feuerloescher nacheinander einsetzen | Bei groeserem Brand: Mehrere Loescher GLEICHZEITIG einsetzen |
