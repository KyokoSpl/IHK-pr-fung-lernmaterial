# Grundlagen: 4.2.05 – Ziele von Betrieben und Unternehmen

## Produktivitaet

**Definition:** Die Produktivitaet misst das Verhaeltnis von Output (Ausbringungsmenge) zu Input (Einsatzmenge). Sie ist eine **mengenbezogene** Kennzahl und gibt an, wie effizient Ressourcen eingesetzt werden.

**Formel:**

```
                     Output (Ausbringungsmenge)
Produktivitaet = ──────────────────────────────────
                     Input (Einsatzmenge)
```

**Kernaussagen:**
- Mengenbezogen: Keine Geldwerte, sondern Stueckzahlen, Stunden, Mengen
- Arbeitsproduktivitaet = Stueckzahl / Arbeitsstunden
- Maschinenproduktivitaet = Stueckzahl / Maschinenstunden
- Steigerung durch: Automation, bessere Werkzeuge, effizientere Prozesse

**Wichtige Begriffe:**

| Begriff | Erklaerung |
|---------|-----------|
| Arbeitsproduktivitaet | Output pro eingesetzte Arbeitsstunde |
| Maschinenproduktivitaet | Output pro eingesetzte Maschinenstunde |
| Kapitalproduktivitaet | Output pro eingesetzter Kapitaleinheit |
| Ausbringungsmenge | Menge der hergestellten Gueter/Dienstleistungen |

**Erklaerung:** Ein IT-Support-Team bearbeitet 200 Tickets in 40 Arbeitsstunden. Die Arbeitsproduktivitaet betraegt: 200 / 40 = **5 Tickets pro Stunde**. Wird ein neues Ticketsystem eingefuehrt und das Team schafft nun 300 Tickets in 40 Stunden (= 7,5 pro Stunde), ist die Produktivitaet gestiegen.

## Rentabilitaet

**Definition:** Die Rentabilitaet misst das Verhaeltnis von Gewinn zum eingesetzten Kapital. Sie gibt an, wie erfolgreich (ertragreich) das eingesetzte Kapital arbeitet. Angabe in **Prozent**.

**Formeln:**

```
                                 Gewinn
Eigenkapitalrentabilitaet = ───────────────── × 100
                              Eigenkapital

                            Gewinn + Fremdkapitalzinsen
Gesamtkapitalrentabilitaet = ─────────────────────────── × 100
                                  Gesamtkapital

                          Gewinn
Umsatzrentabilitaet = ─────────── × 100
                         Umsatz
```

**Kernaussagen:**
- Eigenkapitalrentabilitaet: Wie gut verzinst sich das eigene Kapital?
- Gesamtkapitalrentabilitaet: Wie gut arbeitet das gesamte eingesetzte Kapital?
- Umsatzrentabilitaet: Wie viel Gewinn bleibt pro Euro Umsatz?
- Je hoeher die Rentabilitaet, desto wirtschaftlicher arbeitet das Unternehmen

**Wichtige Begriffe:**

| Begriff | Erklaerung |
|---------|-----------|
| Eigenkapital | Von den Eigentuemern eingebrachtes Kapital |
| Fremdkapital | Geliehenes Kapital (Kredite, Darlehen) |
| Gesamtkapital | Eigenkapital + Fremdkapital |
| Gewinn | Ertrag minus Aufwand (Ueberschuss) |
| Umsatz | Gesamteinnahmen aus dem Verkauf von Guetern/Dienstleistungen |

**Erklaerung:** Ein IT-Unternehmen hat ein Eigenkapital von 200.000 EUR und erzielt einen Jahresgewinn von 30.000 EUR. Die Eigenkapitalrentabilitaet betraegt: (30.000 / 200.000) × 100 = **15 %**. Das bedeutet: Jeder eingesetzte Euro Eigenkapital hat 15 Cent Gewinn erwirtschaftet.

## Wirtschaftlichkeit

**Definition:** Die Wirtschaftlichkeit misst das Verhaeltnis von Ertrag zu Aufwand. Sie ist eine **wertbezogene** Kennzahl (in Geldeinheiten) und gibt an, ob ein Unternehmen gewinnbringend arbeitet.

**Formel:**

```
                          Ertrag (in EUR)
Wirtschaftlichkeit = ─────────────────────────
                          Aufwand (in EUR)
```

**Kernaussagen:**
- Wirtschaftlichkeit > 1: Gewinn (Ertrag groesser als Aufwand)
- Wirtschaftlichkeit = 1: Weder Gewinn noch Verlust
- Wirtschaftlichkeit < 1: Verlust (Aufwand groesser als Ertrag)
- Eng verknuepft mit dem **oekonomischen Prinzip**

**Oekonomisches Prinzip:**

| Prinzip | Beschreibung | Beispiel |
|---------|-------------|---------|
| Maximalprinzip | Mit gegebenen Mitteln den groessten Ertrag erzielen | Mit Budget von 50.000 EUR moeglichst viele Features entwickeln |
| Minimalprinzip | Ein gegebenes Ziel mit geringstem Mitteleinsatz erreichen | Ein bestimmtes Feature mit moeglichst wenig Aufwand umsetzen |
| Optimalprinzip | Bestmoegliches Verhaeltnis von Ertrag zu Aufwand | Bestes Kosten-Nutzen-Verhaeltnis bei der Serverwahl |

**Erklaerung:** Ein IT-Projekt erzielt einen Ertrag von 120.000 EUR bei einem Aufwand von 100.000 EUR. Die Wirtschaftlichkeit betraegt: 120.000 / 100.000 = **1,2**. Das Projekt ist wirtschaftlich (Wert > 1).

## Zielkonflikte

**Definition:** Unternehmen verfolgen verschiedene Ziele gleichzeitig. Diese Ziele koennen sich gegenseitig foerdern (Harmonie), widersprechen (Konflikt) oder unabhaengig voneinander sein (Indifferenz).

**Kernaussagen:**

| Zielbeziehung | Beschreibung | Beispiel |
|---------------|-------------|---------|
| Zielharmonie | Ziel A foerdert Ziel B | Qualitaetssteigerung → hoehere Kundenzufriedenheit |
| Zielkonflikt | Ziel A behindert Ziel B | Kostensenkung vs. hohe Mitarbeiterzufriedenheit |
| Zielindifferenz | Ziele beeinflussen sich nicht | Marketingkampagne hat keinen Einfluss auf Serveruptime |

**Typische Zielkonflikte in IT-Unternehmen:**
- **Sicherheit vs. Benutzerfreundlichkeit:** Strenge Passwortrichtlinien vs. einfache Bedienung
- **Kosten vs. Qualitaet:** Guenstige Hardware vs. hohe Zuverlaessigkeit
- **Schnelligkeit vs. Gruendlichkeit:** Schnelles Release vs. umfassende Tests
- **Datenschutz vs. Funktionsumfang:** Datenminimierung vs. personalisierte Features

**Wichtige Begriffe:**

| Begriff | Erklaerung |
|---------|-----------|
| Oberziel | Uebergeordnetes Unternehmensziel (z.B. Gewinnmaximierung) |
| Unterziel | Ziel, das dem Oberziel dient (z.B. Umsatzsteigerung) |
| Sachziel | Konkretes Leistungsziel (z.B. 1000 Lizenzen verkaufen) |
| Formalziel | Uebergeordnetes Erfolgsziel (z.B. Rentabilitaet steigern) |
| Trade-off | Abwaegung zwischen konkurrierenden Zielen |
