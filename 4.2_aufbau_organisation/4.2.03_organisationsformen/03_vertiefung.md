# Vertiefung: 4.2.03 – Unternehmensstruktur und Organisationsform

## Aufbauorganisation vs. Ablauforganisation

| Kriterium | Aufbauorganisation | Ablauforganisation |
|-----------|-------------------|-------------------|
| Fragestellung | WER macht WAS? | WIE und WANN wird es gemacht? |
| Darstellung | Organigramm | Flussdiagramm, Prozessbeschreibung |
| Fokus | Hierarchie, Stellen, Abteilungen | Reihenfolge, Zeitablauf, Zustaendigkeiten |
| Ziel | Klare Zuordnung von Aufgaben | Effiziente Ablaeufe, kurze Durchlaufzeiten |
| Beispiel | IT-Abteilung mit 3 Teams | Ablauf einer Softwarebereitstellung |

---

## Organisationsformen im Vergleich

### Einliniensystem

```
                +------------------+
                | Geschaeftsfuehrung|
                +------------------+
               /         |          \
              v          v           v
     +---------+  +----------+  +----------+
     | Vertrieb|  | Entwickl.|  |  Support |
     +---------+  +----------+  +----------+
         |             |              |
         v             v              v
     +-------+    +--------+    +---------+
     | MA 1  |    |  MA 2  |    |  MA 3   |
     +-------+    +--------+    +---------+

→ Jeder MA hat genau EINEN Vorgesetzten
→ Klare Verantwortung, aber lange Dienstwege
```

### Mehrliniensystem

```
          +----------+     +----------+
          | Leiter   |     | Leiter   |
          | Qualitaet|     | Technik  |
          +----------+     +----------+
               \              /
                \            /
                 v          v
              +----------------+
              | Mitarbeiter A  |
              +----------------+

→ MA bekommt Weisungen von MEHREREN Vorgesetzten
→ Schnelle Fachkommunikation, aber Konflikte moeglich
```

### Stab-Liniensystem

```
                +------------------+
                | Geschaeftsfuehrung|----[ Stab: Recht ]
                +------------------+----[ Stab: IT-Sicherheit ]
               /         |          \
              v          v           v
     +---------+  +----------+  +----------+
     | Vertrieb|  | Entwickl.|  |  Support |
     +---------+  +----------+  +----------+

→ Stabsstellen beraten die Geschaeftsfuehrung
→ Stabsstellen haben KEIN Weisungsrecht
→ Entlastung der Leitung durch Expertenwissen
```

### Matrixorganisation

```
                      | Produkt A | Produkt B | Produkt C |
    +-----------------+-----------+-----------+-----------+
    | Entwicklung     |   MA 1    |   MA 4    |   MA 7    |
    +-----------------+-----------+-----------+-----------+
    | Vertrieb        |   MA 2    |   MA 5    |   MA 8    |
    +-----------------+-----------+-----------+-----------+
    | Support         |   MA 3    |   MA 6    |   MA 9    |
    +-----------------+-----------+-----------+-----------+

→ MA hat ZWEI Vorgesetzte: Funktionsleiter UND Produktleiter
→ Hohe Flexibilitaet, aber hoher Abstimmungsbedarf
```

### Spartenorganisation

```
                +------------------+
                | Unternehmensltg. |
                +------------------+
               /         |          \
              v          v           v
     +-----------+  +-----------+  +-----------+
     | Sparte A  |  | Sparte B  |  | Sparte C  |
     | (Software)|  | (Hardware)|  | (Cloud)   |
     +-----------+  +-----------+  +-----------+
     | Entwickl. |  | Entwickl. |  | Entwickl. |
     | Vertrieb  |  | Vertrieb  |  | Vertrieb  |
     | Support   |  | Support   |  | Support   |
     +-----------+  +-----------+  +-----------+

→ Jede Sparte hat eigene Funktionsbereiche
→ "Unternehmen im Unternehmen"
→ Marktnaehere Entscheidungen, aber Doppelstrukturen
```

---

## Vor- und Nachteile im Vergleich

| Organisationsform | Vorteile | Nachteile |
|-------------------|---------|-----------|
| Einliniensystem | Klare Zuordnung, einfache Struktur, eindeutige Verantwortung | Lange Dienstwege, Ueberlastung der Leitung, wenig Flexibilitaet |
| Mehrliniensystem | Kurze Wege, spezialisierte Anweisungen | Widerspruchliche Weisungen, unklare Zustaendigkeiten |
| Stab-Liniensystem | Entlastung der Leitung, Nutzung von Expertenwissen | Konflikte zwischen Stab und Linie, informelle Machtbildung |
| Matrixorganisation | Hohe Flexibilitaet, interdisziplinaere Zusammenarbeit | Kompetenzueberschneidungen, hoher Abstimmungsbedarf |
| Spartenorganisation | Marktnaehe, Ergebnistransparenz pro Sparte | Doppelstrukturen, hoehere Kosten, Spartenegoismus |

---

## IT-Abteilung in der Organisation

Die IT-Abteilung kann in verschiedenen Positionen eingeordnet werden:

| Variante | Beschreibung | Vorteil |
|----------|-------------|---------|
| Stabsstelle der GF | IT beraet die Geschaeftsfuehrung direkt | Strategische Naehe, kurze Wege zur Leitung |
| Eigene Abteilung (Linie) | IT als gleichberechtigte Abteilung neben Vertrieb, Produktion etc. | Klare Struktur, eigene Ressourcen |
| Teil einer Sparte | Jede Sparte hat eigene IT | Naehe zum Fachbereich, spezifisches Know-how |
| Zentralbereich | IT als uebergreifende Serviceeinheit | Einheitliche Standards, Kosteneffizienz |

## Typische Pruefungsaspekte
- Organigramm zeichnen oder vervollstaendigen
- Organisationsform anhand eines Organigramms identifizieren
- Vor- und Nachteile einer Organisationsform benennen
- IT-Abteilung in ein Organigramm einordnen
- Unterscheidung Aufbau- vs. Ablauforganisation

## Haeufige Fehler

| Fehler | Korrektur |
|--------|-----------|
| Stabsstelle hat Weisungsrecht | Stabsstellen beraten NUR, kein Weisungsrecht |
| Matrix = Einliniensystem mit zwei Ebenen | Matrix = ZWEI Dimensionen gleichzeitig (Funktion + Produkt) |
| Spartenorganisation = nur grosse Konzerne | Auch mittelstaendische Unternehmen nutzen Spartenorganisation |
| Aufbau- und Ablauforg. sind dasselbe | Aufbau = WER, Ablauf = WIE/WANN – sie ergaenzen sich |
