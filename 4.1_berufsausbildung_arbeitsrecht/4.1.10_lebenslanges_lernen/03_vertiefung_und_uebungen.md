# Vertiefung und Uebungen: 4.1.10 – Lebensbegleitendes Lernen

## Vertiefung

### Vergleich: Weiterbildungsformen

| Kriterium | Innerbetrieblich | Ausserbetrieblich | Selbststudium |
|-----------|-----------------|-------------------|---------------|
| Organisation | Arbeitgeber | Externer Anbieter | Eigeninitiative |
| Kosten fuer AN | Meist kostenfrei | Teilweise AN-Anteil | AN traegt Kosten |
| Praxisbezug | Hoch (betriebsspezifisch) | Mittel (allgemein) | Variabel |
| Zertifikat | Selten (intern) | Oft (IHK, Hersteller) | Moeglich (Online) |
| Zeitpunkt | Waehrend Arbeitszeit | Arbeitszeit oder Freizeit | Freizeit |
| Beispiel | Onboarding, internes Seminar | IHK-Lehrgang, Herstellerschulung | Online-Kurs, Fachbuch |

### Vergleich: Fortbildung vs. Umschulung (Detail)

```
Fortbildung                          Umschulung
+-------------------------------+    +-------------------------------+
| Ziel: Im bestehenden Beruf   |    | Ziel: Neuer Beruf            |
| bleiben & weiterentwickeln    |    |                               |
+-------------------------------+    +-------------------------------+
| Arten:                        |    | Ausloeser:                   |
| - Erhaltung                   |    | - Arbeitslosigkeit            |
| - Anpassung                   |    | - Gesundheitliche Gruende     |
| - Aufstieg                    |    | - Strukturwandel              |
+-------------------------------+    +-------------------------------+
| Dauer: Stunden bis Monate     |    | Dauer: 1-2 Jahre             |
+-------------------------------+    +-------------------------------+
| Abschluss: Zertifikat         |    | Abschluss: Berufsabschluss   |
+-------------------------------+    +-------------------------------+
| Foerderung: Aufstiegs-BAFoeG  |    | Foerderung: Bildungsgutschein |
| Bildungspraemie               |    | Agentur fuer Arbeit          |
+-------------------------------+    +-------------------------------+
```

```
+---------------------------------------------------+
```

### Typische Pruefungsaspekte

- Unterschied Fortbildung vs. Umschulung erlaeutern
- Foerdermassnahme zu einem Szenario zuordnen (z.B. arbeitslos → Bildungsgutschein)
- Bedeutung des lebenslangen Lernens in der IT-Branche begruenden
- Bildungsurlaub erklaeren (Anspruch, Dauer, Voraussetzungen)
- Innerbetriebliche Fortbildungsformen (On-the-Job, Near-the-Job, Off-the-Job) unterscheiden

### Haeufige Fehler

| Fehler | Richtigstellung |
|--------|----------------|
| Fortbildung = Umschulung | Falsch – Fortbildung baut auf bestehendem Beruf auf, Umschulung fuehrt in neuen Beruf |
| Bildungsurlaub gibt es ueberall | Falsch – Bayern und Sachsen haben kein eigenes Bildungsurlaubsgesetz |
| AG muss jede Weiterbildung bezahlen | Falsch – nur wenn tarifvertraglich/betrieblich vereinbart oder gesetzlich geregelt |
| Aufstiegs-BAFoeG ist nur fuer Handwerker | Falsch – auch fuer IT-Fachkraefte bei Aufstiegsfortbildung (z.B. Operative Professional) |
| Lebenslanges Lernen ist freiwillig | In der IT praktisch Pflicht – Technologien aendern sich zu schnell |

## Uebungen

**Aufgabe 1:** Erklaere den Unterschied zwischen Fortbildung und Umschulung nach BBiG. Nenne je ein Beispiel aus der IT-Branche.

**Aufgabe 2:** Ordne folgende Weiterbildungsformen der richtigen Kategorie zu (On-the-Job / Near-the-Job / Off-the-Job):

# Loesungen

## Aufgabe 1
- **Fortbildung (§ 1 Abs. 4 BBiG):** Baut auf bestehender Ausbildung auf. Erweitert, vertieft oder erneuert Kenntnisse im aktuellen Berufsfeld. Beispiel: Ein Fachinformatiker macht eine ITIL-Zertifizierung.
- **Umschulung (§ 1 Abs. 5 BBiG):** Fuehrt in einen voellig neuen Beruf. Beispiel: Ein Einzelhandelskaufmann wird zum Fachinformatiker fuer Anwendungsentwicklung umgeschult.

## Aufgabe 2
- a) Einarbeitung → **On-the-Job** (direkt am Arbeitsplatz)
- b) Externes Seminar → **Off-the-Job** (ausserhalb des Arbeitsplatzes)
- c) Qualitaetszirkel → **Near-the-Job** (in Arbeitsplatznaehe, aber nicht regulaere Taetigkeit)
- d) E-Learning → **Off-the-Job** (unabhaengig vom Arbeitsplatz, kann auch Near-the-Job sein, wenn waehrend der Arbeitszeit)

## Aufgabe 3
**Aufstiegs-BAFoeG (Meister-BAFoeG).** Begruendung: Der IT-Projektleiter (Operative Professional) ist eine anerkannte Aufstiegsfortbildung mit IHK-Pruefung. Das Aufstiegs-BAFoeG foerdert Lehrgangs- und Pruefungskosten durch Zuschuss und zinsguentiges Darlehen.

## Aufgabe 4
1. Technologien in der IT veralten schnell (neue Programmiersprachen, Frameworks, Cloud-Services)
2. Neue Berufsbilder entstehen (z.B. Data Engineer, DevOps Engineer, KI-Spezialist)
3. Gesetzliche Anforderungen aendern sich (DSGVO, IT-Sicherheitsgesetz, NIS2) und erfordern aktuelles Wissen

## Aufgabe 5
Bildungsurlaub ist ein gesetzlicher Anspruch auf bezahlte Freistellung fuer anerkannte Weiterbildungsveranstaltungen. In der Regel stehen 5 Arbeitstage pro Jahr zur Verfuegung. Bayern und Sachsen haben kein eigenes Bildungsurlaubsgesetz.

## Aufgabe 6
**Umschulung** – Die Mitarbeiterin wechselt den Beruf komplett (von Burokauffrau zu Fachinformatikerin). Eine Fortbildung wuerde nicht ausreichen, da es sich um ein voellig neues Berufsfeld handelt. Passende Foerderung: **Bildungsgutschein** der Agentur fuer Arbeit (wenn Arbeitslosigkeit droht oder vorliegt) oder Foerderung ueber das **Qualifizierungschancengesetz** (wenn sie noch beschaeftigt ist und vom Strukturwandel betroffen).
