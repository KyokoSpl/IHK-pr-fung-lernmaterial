# Grundlagen: 4.1.14 – Lebensplanung

## Karriereplanung

**Definition:** Karriereplanung ist die systematische Gestaltung des beruflichen Werdegangs durch Zielsetzung, Kompetenzentwicklung und strategische Entscheidungen.

### Karrierewege in der IT-Branche

```
Einstieg            Erfahren           Senior              Experte/Fuehrung
+-----------+    +-----------+    +---------------+    +------------------+
| Junior    | -> | Developer | -> | Senior Dev    | -> | Lead / Architekt |
| Developer |    | (Mid)     |    | / Engineer    |    | / CTO            |
+-----------+    +-----------+    +---------------+    +------------------+
     |                |                  |                      |
     v                v                  v                      v
 0-2 Jahre        2-5 Jahre          5-10 Jahre            10+ Jahre
```

**Alternative Karrierepfade:**

| Pfad | Beschreibung | Beispielpositionen |
|------|-------------|-------------------|
| Fachkarriere | Vertiefte technische Spezialisierung | Architekt, Principal Engineer, Security Expert |
| Fuehrungskarriere | Teamleitung, Abteilungsleitung | Teamlead, Head of IT, CTO |
| Projektkarriere | Projektleitung und -steuerung | Projektleiter, Scrum Master, Product Owner |
| Beratungskarriere | Externe oder interne Beratung | IT-Consultant, Freiberufler |
| Gruenderkarriere | Eigenes Unternehmen / Start-up | CEO, Geschaeftsfuehrer |

### SMART-Ziele fuer die Karriere
- **S**pezifisch: "Ich moechte AWS Solutions Architect werden"
- **M**essbar: "Ich bestehe die Zertifizierungspruefung"
- **A**kzeptiert: "Mein Arbeitgeber unterstuetzt die Fortbildung"
- **R**ealistisch: "Ich habe bereits Cloud-Erfahrung"
- **T**erminiert: "Bis Ende Q2 2027"

## Potenzialanalyse

**Definition:** Die Potenzialanalyse ist eine systematische Bewertung der eigenen Staerken, Schwaechen, Faehigkeiten und Entwicklungsmoeglichkeiten.

**Methoden:**

| Methode | Beschreibung |
|---------|-------------|
| SWOT-Analyse (persoenlich) | Staerken, Schwaechen, Chancen, Risiken der eigenen Karriere |
| Kompetenzprofil | Erfassung fachlicher, methodischer, sozialer und persoenlicher Kompetenzen |
| 360-Grad-Feedback | Feedback von Vorgesetzten, Kollegen, Untergebenen, Kunden |
| Selbsteinschaetzung | Eigene Bewertung der Faehigkeiten und Interessen |
| Assessment-Center | Strukturierte Uebungen zur Bewertung von Kompetenzen |
| Mitarbeitergespraech | Regelmaessiges Gespraech mit Vorgesetztem ueber Ziele und Entwicklung |

**SWOT-Analyse (persoenlich):**

```
              Positiv              Negativ
         +------------------+------------------+
Intern   | Staerken          | Schwaechen       |
         | (Strengths)       | (Weaknesses)     |
         | z.B. Gute Python- | z.B. Wenig       |
         | Kenntnisse        | Cloud-Erfahrung  |
         +------------------+------------------+
Extern   | Chancen           | Risiken          |
         | (Opportunities)   | (Threats)        |
         | z.B. Wachsender   | z.B. KI ersetzt  |
         | Cloud-Markt       | einfache Aufgaben|
         +------------------+------------------+
```

## Europass

**Definition:** Der Europass ist ein EU-weit standardisiertes Dokumentenpaket, das Qualifikationen und Kompetenzen europaweit vergleichbar und verstaendlich macht.

**Bestandteile:**

| Dokument | Beschreibung |
|----------|-------------|
| Europass-Lebenslauf | Standardisiertes Lebenslauf-Format (online erstellbar) |
| Europass-Sprachenpass | Selbsteinschaetzung der Sprachkenntnisse (A1–C2) |
| Europass-Mobilitaet | Nachweis ueber Auslandsaufenthalte und Praktika |
| Diploma Supplement | Ergaenzung zu Hochschulabschluessen |
| Zeugniserlaeuterung | Erklaerung beruflicher Abschluesse fuer das Ausland |

**Website:** europass.cedefop.europa.eu (kostenlos)

## Auslandsaufenthalte und Mobilitaetsprogramme

**Definition:** Programme, die berufliche und bildungsbezogene Aufenthalte im Ausland foerdern.

### Erasmus+

| Merkmal | Beschreibung |
|---------|-------------|
| Traeger | Europaeische Union |
| Zielgruppe | Auszubildende, Studierende, Berufstaetige, Lehrkraefte |
| Leistung | Zuschuss fuer Reise, Unterkunft, Lebenshaltung |
| Dauer | 2 Wochen bis 12 Monate (je nach Programmlinie) |
| Ziel | Berufliche/persoenliche Erfahrung im EU-Ausland |
| IT-Bezug | Praktikum bei IT-Unternehmen im Ausland |

**Weitere Programme:**
- **EuroApprenticeships:** Auslandspraktika fuer Auszubildende
- **Leonardo da Vinci** (jetzt Teil von Erasmus+): Berufliche Bildung
- **EURES:** EU-Jobportal fuer grenzueberschreitende Arbeitsvermittlung

## Europaeische Sozialcharta

**Definition:** Die Europaeische Sozialcharta (Europarat, 1961/revidiert 1996) legt soziale Grundrechte fest, darunter:

- Recht auf Arbeit und gerechte Arbeitsbedingungen
- Recht auf ein gerechtes Arbeitsentgelt
- Recht auf Koalitionsfreiheit
- Recht auf Berufsausbildung und berufliche Weiterbildung
- Recht auf sozialen Schutz (Sozialversicherung)
- Recht auf Schutz von Kindern und Jugendlichen

**Pruefungsrelevanz:** Die Charta zeigt, dass soziale Rechte europaweit verankert sind und nicht nur national geregelt werden.

## Existenzgruendung

**Definition:** Existenzgruendung ist die Gruendung eines eigenen Unternehmens als selbststaendige Erwerbstaetigkeit.

### Schritte der Existenzgruendung

1. **Geschaeftsidee entwickeln** (z.B. IT-Dienstleistung, Software-Produkt)
2. **Marktanalyse** (Zielgruppe, Wettbewerb, Marktnachfrage)
3. **Businessplan erstellen** (siehe unten)
4. **Rechtsform waehlen** (Einzelunternehmen, GbR, GmbH, UG)
5. **Finanzierung sichern** (Eigenkapital, Kredit, Foerdermittel)
6. **Gewerbeanmeldung / Freiberuflerstauts klaren**
7. **Versicherungen abschliessen** (KV, Haftpflicht, Betriebshaftpflicht)
8. **Kunden akquirieren und starten**

### Businessplan

| Bestandteil | Inhalt |
|------------|--------|
| Executive Summary | Kurzzusammenfassung der Geschaeftsidee |
| Geschaeftsidee | Produkt/Dienstleistung, Alleinstellungsmerkmal (USP) |
| Marktanalyse | Zielgruppe, Marktgroesse, Wettbewerber |
| Marketing | Vertriebsstrategie, Preisgestaltung |
| Organisation | Rechtsform, Team, Standort |
| Finanzplan | Umsatzprognose, Kosten, Kapitalbedarf, Liquiditaet |
| Risikoanalyse | Chancen und Risiken, Gegenstrategien |

### Rechtsformen

| Rechtsform | Haftung | Mindestkapital | Gruender |
|-----------|---------|---------------|---------|
| Einzelunternehmen | Unbeschraenkt (Privatvermoegen) | Keins | 1 Person |
| GbR | Unbeschraenkt (alle Gesellschafter) | Keins | Min. 2 Personen |
| UG (haftungsbeschraenkt) | Beschraenkt auf Gesellschaftsvermoegen | Ab 1 EUR | Min. 1 Person |
| GmbH | Beschraenkt auf Gesellschaftsvermoegen | 25.000 EUR | Min. 1 Person |

### Chancen und Grenzen

| Chancen | Grenzen/Risiken |
|---------|----------------|
| Selbstbestimmung, eigene Ideen umsetzen | Hohes finanzielles Risiko |
| Unbegrenztes Einkommenspotenzial | Kein festes Gehalt, Einkommensunsicherheit |
| Flexible Arbeitszeiten | Lange Arbeitszeiten, Verantwortungsdruck |
| Steuerliche Vorteile moeglich | Komplexe Buchfuehrung und Steuerpflichten |
| IT-Branche: Hohe Nachfrage nach Freelancern | Kundenakquise, Wettbewerbsdruck |

## Kompetenzbereiche der persoenlichen Entwicklung

| Kompetenz | Beschreibung | IT-Beispiel |
|-----------|-------------|-------------|
| Eigenstaendigkeit | Selbstorganisiertes Arbeiten | Selbststaendig Code schreiben und testen |
| Verantwortung | Zuverlaessigkeit, Rechenschaft | Produktivsystem betreuen, Fehler verantworten |
| Reflexivitaet | Eigenes Handeln kritisch hinterfragen | Code-Review annehmen, Lessons Learned |
| Lernkompetenz | Faehigkeit, sich neues Wissen anzueignen | Neue Technologie selbststaendig erlernen |
| Teamfaehigkeit | Kooperatives Arbeiten | Agile Teamarbeit, Pair Programming |
| Fuehrungsfaehigkeit | Andere anleiten und motivieren | Teamleitung, Mentoring von Azubis |
| Mitgestaltung | Aktiv an Verbesserungen mitwirken | Prozessoptimierung vorschlagen |
| Kommunikation | Klar und adressatengerecht kommunizieren | Technische Sachverhalte fuer Nicht-Techniker erklaeren |

## Persoenliche Weiterbildung

**Moeglichkeiten:**
- Fachliteratur lesen (Buecher, Fachartikel, Blogs)
- Fachmessen besuchen (CeBIT, it-sa, re:publica)
- Online-Selbstlernmaterialien (MOOCs, Tutorials)
- Open-Source-Projekte beitragen (GitHub)
- Netzwerken (Meetups, Konferenzen, LinkedIn)
- Berufsbegleitendes Studium / Fernstudium

## Ueberbetriebliche Fortbildung

**Definition:** Fortbildungsmassnahmen, die ausserhalb des eigenen Betriebs stattfinden and von externen Traegern durchgefuehrt werden.

| Traeger | Beispiele |
|---------|----------|
| IHK | Vorbereitungskurse Operative Professional, Seminare |
| Private Bildungstraeger | WBS Training, GFN, Comcave |
| Hochschulen / Fernunis | Berufsbegleitendes Studium, Zertifikatskurse |
| Hersteller | Microsoft Learning, AWS Training, Cisco Academy |
| Verbandsakademien | Bitkom Akademie, VDI-Kurse |

## Bildungseinrichtungen

| Einrichtung | Angebot | Besonderheit |
|------------|--------|-------------|
| IHK (Industrie- und Handelskammer) | Pruefungen, Lehrgaenge, Beratung | Zustaendig fuer Berufsausbildung und Fortbildungspruefungen |
| Hochschulen (Uni/FH) | Bachelor, Master, Promotion | Akademische Laufbahn |
| Fernuniversitaet (z.B. FernUni Hagen) | Berufsbegleitendes Studium | Flexibel, ortsunabhaengig |
| Volkshochschulen (VHS) | Allgemeine und berufliche Weiterbildung | Guenstig, niedrigschwellig |
| Private Bildungsinstitute | Umschulungen, Zertifizierungskurse | Oft AZAV-zertifiziert fuer Bildungsgutschein |
| Berufsschulen | Ausbildungsbegleitend, Zusatzqualifikationen | Duale Ausbildung |
