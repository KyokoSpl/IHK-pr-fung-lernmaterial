# Ueberblick: 3.4.22 – Datenbankabfrage, Datenpflege mit SQL

## Einordnung

- **Pruefungsteil:** 3.4 – Konzipieren und Umsetzen von kundenspezifischen Softwareanwendungen
- **Thema-ID:** 3.4.22
- **Thema:** Datenbankabfrage, Datenpflege mit SQL

## Ziel

Du musst SQL-Abfragen schreiben und verstehen koennen: Von einfachen SELECT-Anweisungen ueber JOINs, Unterabfragen und Aggregatfunktionen bis zu DDL-Anweisungen (CREATE TABLE, ALTER TABLE, CREATE INDEX). SQL ist eines der am staerksten pruefungsrelevanten Themen der IHK-Pruefung.

## Themenkreise im Ueberblick

### 1. Projektion (SELECT FROM)
Auswahl bestimmter Spalten aus einer Tabelle. Bestimmt, WELCHE Spalten in der Ergebnismenge erscheinen.

### 2. Selektion (SELECT FROM WHERE) und verschachtelte SELECTs
Auswahl bestimmter Zeilen durch Filterbedingungen. Unterabfragen (Subqueries) ermoeglichen verschachtelte Abfragen.

### 3. Ausdruecke/Bedingungen
Vergleichsoperatoren, logische Operatoren (AND, OR, NOT), BETWEEN, IN, LIKE, IS NULL.

### 4. Sortieren (ORDER BY)
Sortierung der Ergebnismenge nach einer oder mehreren Spalten, aufsteigend (ASC) oder absteigend (DESC).

### 5. Aggregatfunktionen
Berechnungen ueber Gruppen von Zeilen: COUNT, SUM, AVG, MIN, MAX.

### 6. Gruppieren (GROUP BY, HAVING)
Zusammenfassung von Zeilen in Gruppen, ggf. mit Filterbedingung auf die Gruppen (HAVING).

### 7. Abfrage ueber mehrere Tabellen
JOINs (INNER JOIN, LEFT JOIN, RIGHT JOIN) zur Verknuepfung von Tabellen ueber Schluesselbeziehungen.

### 8. Manipulation (INSERT, UPDATE, DELETE)
Einfuegen, Aendern und Loeschen von Datensaetzen (DML – Data Manipulation Language).

### 9. Tabellenstruktur (CREATE TABLE, ALTER TABLE)
Erstellen und Aendern von Tabellen inkl. Datentypen, Primaer-/Fremdschluessel und Constraints (DDL – Data Definition Language).

### 10. Index (CREATE INDEX)
Erstellen von Indizes zur Beschleunigung von Abfragen.
