## Dateistruktur

Jedes Thema hat je nach Anzahl der Themenkreise (TK) eine unterschiedliche Dateistruktur:

```
5+ Themenkreise → 5 Dateien:
  01_ueberblick.md        Einordnung, Ziele, Themenkreis-Tabelle
  02_grundlagen.md        Definitionen, Fachbegriffe, Tabellen
  03_vertiefung.md        Vergleiche, Diagramme, Pruefungsaspekte
  04_beispiele.md         Praxisnahe Szenarien
  05_uebungen.md          Aufgaben + Loesungen

3–4 Themenkreise → 3 Dateien:
  01_ueberblick.md
  02_grundlagen.md
  03_vertiefung_und_uebungen.md

1–2 Themenkreise → 1 Datei:
  01_komplett.md          Alles in einer Datei
```

## Hinweise

- Alle Inhalte verwenden **Umlaut-Ersetzungen** (ae, oe, ue, ss) fuer maximale Kompatibilitaet
- Loesungen zu Uebungsaufgaben sind durch `---` vom Aufgabenteil getrennt
- Die Themen orientieren sich am **offiziellen IHK-Pruefungskatalog** fuer Fachinformatiker/in Anwendungsentwicklung
- Kein Anspruch auf Vollstaendigkeit oder Fehlerfreiheit – immer mit offiziellen Quellen abgleichen!

## Nutzung

Die Dateien sind reines Markdown und koennen mit jedem Texteditor, Obsidian, VS Code oder direkt auf GitHub gelesen werden.

> **Tipp:** In Obsidian lassen sich die Dateien als Vault oeffnen fuer eine verlinkte Lernumgebung.
