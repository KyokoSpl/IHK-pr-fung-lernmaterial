## Aktives Zuhoeren

**Definition:** Gespraechstechnik, bei der der Zuhoerer dem Gespraechspartner signalisiert, dass er aufmerksam zuhoert, versteht und sich einfuehlt.

**Techniken:**

| Technik | Beschreibung | Beispiel |
|---|---|---|
| Paraphrasieren | Inhalt in eigenen Worten wiedergeben | "Wenn ich Sie richtig verstehe, moechten Sie..." |
| Spiegeln | Gefuehle des Gegenueber benennen | "Das scheint Sie zu frustrieren." |
| Nachfragen | Offene Fragen stellen | "Koennten Sie das naeher erlaeutern?" |
| Zusammenfassen | Kernpunkte buendeln | "Also zusammengefasst geht es um..." |
| Nonverbale Signale | Nicken, Blickkontakt, offene Haltung | Zugewandte Koerperhaltung |

## Kommunikationsmodelle

### Sender-Empfaenger-Modell (Shannon/Weaver)

```
+--------+    Nachricht    +-----------+
| Sender |---------------->| Empfaenger|
+--------+    (Kanal)      +-----------+
     ^                          |
     |      Feedback            |
     +--------------------------+

Stoerungen koennen auf dem Kanal auftreten
```

### 4-Ohren-Modell (Schulz von Thun)

Jede Nachricht hat vier Ebenen:

| Ebene | Beschreibung | Beispiel: "Die Ampel ist gruen." |
|---|---|---|
| Sachinhalt | Worüber informiert wird | Die Ampel zeigt gruen |
| Selbstoffenbarung | Was der Sender ueber sich verraet | "Ich habe es eilig" |
| Beziehung | Wie Sender zum Empfaenger steht | "Du brauchst meine Hilfe" |
| Appell | Was der Sender bewirken will | "Fahr los!" |

### Eisbergmodell

```
        /\        ← Sachebene (20%)
       /  \          Fakten, Daten, sichtbares Verhalten
      /    \
     /      \
    /--------\
   /          \   ← Beziehungsebene (80%)
  /            \     Gefuehle, Werte, Beduerfnisse,
 /              \    Erfahrungen, Aengste
/________________\
```

## Verkaufsgespraeche

### Phasen eines Verkaufsgespraechs

| Phase | Inhalt | Technik |
|---|---|---|
| 1. Kontaktaufnahme | Begruessing, Vertrauen aufbauen | Small Talk, offene Fragen |
| 2. Bedarfsanalyse | Kundenbeduerfnisse ermitteln | Aktives Zuhoeren, W-Fragen |
| 3. Praesentation | Loesung vorstellen | Nutzenargumentation |
| 4. Einwandbehandlung | Bedenken ausraeumen | Ja-aber-Methode, Bumerang-Methode |
| 5. Abschluss | Kaufentscheidung herbeifuehren | Zusammenfassung, Alternativfrage |
| 6. Nachbereitung | Dokumentation, Follow-up | CRM-Eintrag, Terminnachverfolgung |

### Anfrage → Angebot → Auftrag

```
Kunde                          Anbieter
  |                               |
  |--- Anfrage ------------------>|
  |   (Bedarfsbeschreibung)       |
  |                               |
  |<-- Angebot -------------------|
  |   (Leistung, Preis, Fristen)  |
  |                               |
  |--- Auftrag ------------------>|
  |   (Annahme = Vertrag)         |
  |                               |
  |<-- Auftragsbestaetigung ------|
```

**Verbindlichkeit:**
- **Anfrage:** Unverbindlich, keine Verpflichtung
- **Angebot:** Bindend fuer den Anbieter (Frist beachten)
- **Auftrag:** Rechtsverbindlicher Vertrag nach Annahme

### Analyse der Kundenbeduerfnisse

**Fragetechniken:**
- **Offene Fragen:** "Was erwarten Sie von der neuen Loesung?"
- **Geschlossene Fragen:** "Benoetigen Sie auch eine mobile Anbindung?"
- **Alternativfragen:** "Bevorzugen Sie Cloud oder On-Premises?"

**Bedarfsermittlung im IT-Kontext:**

| Aspekt | Typische Fragen |
|---|---|
| Ist-Zustand | Welche Systeme nutzen Sie aktuell? |
| Soll-Zustand | Was soll die neue Loesung koennen? |
| Budget | In welchem Rahmen bewegt sich Ihr Budget? |
| Zeitrahmen | Bis wann soll die Loesung produktiv sein? |
| Nutzeranzahl | Wie viele Mitarbeiter arbeiten damit? |
| Schnittstellen | Welche bestehenden Systeme muessen angebunden werden? |

## Uebungen

### Aufgabe 1: 4-Ohren-Modell anwenden
Ein Kunde sagt: "Das haetten wir auch selbst machen koennen."
Interpretiere diese Aussage auf allen vier Ebenen des 4-Ohren-Modells.

---
---

**Loesung Aufgabe 1:**
- **Sachinhalt:** Wir haetten die Faehigkeit gehabt, die Aufgabe selbst auszufuehren.
- **Selbstoffenbarung:** Ich bin unsicher, ob der Auftrag sein Geld wert war / Ich fuehle mich ueberlegen.
- **Beziehung:** Ich traue Ihnen nicht mehr Kompetenz zu als uns selbst.
- **Appell:** Rechtfertigen Sie den Mehrwert Ihrer Leistung / Senken Sie den Preis.

### Aufgabe 2: Verkaufsgespraech
Ein mittelstaendisches Unternehmen mit 50 Mitarbeitern fragt nach einer neuen E-Mail-Loesung. Der aktuelle Mailserver ist veraltet und faellt regelmaessig aus.
a) Formuliere drei offene Fragen fuer die Bedarfsanalyse.
b) Nenne drei Punkte, die das Angebot enthalten sollte.

---
---

**Loesung Aufgabe 2:**
a) (1) "Welche Probleme treten mit der aktuellen Loesung am haeufigsten auf?" (2) "Welche Funktionen sind Ihren Mitarbeitern besonders wichtig, z.B. Kalender, Kontakte, mobiler Zugriff?" (3) "Gibt es bestehende Systeme wie Active Directory oder CRM, die angebunden werden muessen?"
b) (1) Leistungsbeschreibung: Migration von 50 Postfaechern, Einrichtung Exchange Online/Microsoft 365. (2) Kosten: Einmalige Migrationskosten + monatliche Lizenzkosten pro Nutzer. (3) Zeitplan: Meilensteine fuer Migration und Schulung mit konkreten Terminen.
