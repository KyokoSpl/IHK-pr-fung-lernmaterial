# Vertiefung: 2.3.02 – Methoden zur Messung der Zielerreichung im QM-Prozess

## Zusammenhang der Themenkreise

```
Qualitaetsplanung (Soll-Werte)
           |
           v
    +--------------+
    | Testplanung  |
    | Testdaten-   |
    | generatoren  |
    +--------------+
           |
           v
    +--------------+
    | Testdurch-   |
    | fuehrung     |
    +--------------+
           |
           v
    +--------------+      +-----------------+
    | Testprotokoll|----->| Soll-Ist-       |
    | (Ist-Werte)  |      | Vergleich       |
    +--------------+      +-----------------+
                                  |
                          +-------+-------+
                          |               |
                   +-----------+    +------------+
                   | Abnahme-  |    | PDCA/KVP   |
                   | protokoll |    | Verbesserung|
                   +-----------+    +------------+
                                         |
                                    Kennzahlen
                                    (KPIs messen)
```

## Vergleich: Abnahmeprotokoll vs. Testprotokoll

| Merkmal | Testprotokoll | Abnahmeprotokoll |
|---|---|---|
| Erstellt von | Tester / QA | Auftraggeber + Auftragnehmer |
| Zeitpunkt | Waehrend Entwicklung | Bei Projektuebergabe |
| Fokus | Technische Funktionalitaet | Vertragserfuellung |
| Verbindlichkeit | Intern | Rechtlich relevant |
| Inhalt | Testfaelle + Ergebnisse | Maengel + Abnahmestatus |

## Typische Pruefungsaspekte

- Abnahmeprotokoll-Inhalte benennen
- Soll-Ist-Abweichungen berechnen (absolut + relativ)
- PDCA-Phasen einem Szenario zuordnen
- Testprotokoll-Felder erlaeutern
- Kennzahlen berechnen und interpretieren
- KVP-Prinzipien erklaeren

## Haeufige Fehler in Pruefungen

| Fehler | Richtig |
|---|---|
| Abnahme = automatisch nach Lieferung | Abnahme muss erklaert werden (foermlich) |
| KVP = einmalige grosse Verbesserung | KVP = kontinuierlich, kleine Schritte |
| Testprotokoll nur bei Fehlern | Auch bestandene Tests dokumentieren |
| Soll-Ist nur fuer Kosten | Auch fuer Zeit, Qualitaet, Verfuegbarkeit |
