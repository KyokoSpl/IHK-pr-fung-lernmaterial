# Grundlagen: 1.7.01 – Vertragsarten und Vertragsstoerungen

## Kaufvertrag, Mietvertrag, Leasing

### Kaufvertrag (§§ 433 ff. BGB)
- Eigentumsuebergang: Verkaeufer uebergibt Sache, Kaeufer zahlt Preis
- IT-Beispiel: Kauf von Hardware, Softwarelizenzen (Dauerlizenz)
- Gewaehrleistung: 2 Jahre (B2C), verhandelbar (B2B)

### Mietvertrag (§§ 535 ff. BGB)
- Gebrauchsueberlassung auf Zeit gegen Entgelt
- IT-Beispiel: Anmietung von Serverraeumen, Hardware-Miete
- Eigentumsuebergang: NEIN – Sache wird zurueckgegeben

### Leasing
- Sonderform der Miete mit Finanzierungscharakter
- IT-Beispiel: Leasing von Notebooks, Druckern
- Optionen am Ende: Rueckgabe, Kauf, Verlaengerung
- Buchhaltung: Leasingraten als Betriebsausgabe

**Vergleich:**

| Kriterium | Kauf | Miete | Leasing |
|---|---|---|---|
| Eigentumsuebergang | Ja | Nein | Meist nein (Option) |
| Einmalzahlung | Ja | Nein (Raten) | Nein (Raten) |
| Gewaehrleistung | Verkaeufer | Vermieter | Leasinggeber |
| Abschreibung | Kaeufer | Vermieter | Leasinggeber |

## Lizenzvertrag

**Definition:** Vertrag ueber das Nutzungsrecht an Software oder geistigem Eigentum.

**Lizenzmodelle:**

| Modell | Beschreibung | Beispiel |
|---|---|---|
| **Einzelplatzlizenz** | 1 Installation auf 1 Geraet | MS Office Home |
| **Volumenlizenz** | Mehrere Lizenzen zu Sonderkonditionen | MS Enterprise Agreement |
| **OEM-Lizenz** | An Hardware gebunden | Windows vorinstalliert |
| **Mietlizenz/SaaS** | Nutzung gegen laufende Gebuehr | Microsoft 365 |
| **Open Source** | Frei nutzbar (verschiedene Lizenzen) | GPL, MIT, Apache |

---

## Servicevertrag / Service-Level-Agreement (SLA)

**Servicevertrag:** Vereinbarung ueber laufende IT-Dienstleistungen (Wartung, Support, Betrieb).

**SLA-Kennzahlen:**

| Kennzahl | Beschreibung | Beispiel |
|---|---|---|
| **Verfuegbarkeit** | Uptime in Prozent | 99,9% = max. 8,76 h Ausfall/Jahr |
| **Reaktionszeit** | Zeit bis erste Reaktion | < 1 Stunde |
| **Wiederherstellungszeit** | Zeit bis zur Loesung | < 4 Stunden |
| **Servicezeiten** | Wann wird Support geleistet? | Mo-Fr 8-18 Uhr / 24/7 |

## Vertragsbestandteile

| Bestandteil | Beschreibung |
|---|---|
| Vertragspartner | Wer schliesst den Vertrag? |
| Leistungsbeschreibung | Was genau wird geschuldet? |
| Termine/Fristen | Wann muss geliefert werden? |
| Entgelte/Preise | Was kostet die Leistung? |
| Zahlungsbedingungen | Wann und wie wird bezahlt? |
| Gewaehrleistung/Haftung | Wer haftet bei Maengeln? |
| Kuendigung | Wie und wann kann gekuendigt werden? |
| Sanktionen/Konventionalstrafen | Strafen bei Nichterfuellung |

---

## Vertragsstoerungen

| Stoerungsart | Beschreibung | Rechtsfolge |
|---|---|---|
| **Schlechtleistung** | Mangelhaft erfuellt | Nachbesserung, Minderung, Ruecktritt |
| **Nichtleistung** | Gar nicht erfuellt | Schadenersatz, Ruecktritt |
| **Verzug** | Zu spaet erfuellt | Mahnung → Frist → Schadenersatz/Ruecktritt |
| **Unmoeglich** | Leistung nicht mehr moeglich | Befreiung von Leistungspflicht, Schadenersatz |

**Ablauf bei Verzug:**
1. Faelligkeit des Termins
2. Mahnung (bei kalenderdefiniertem Termin entbehrlich)
3. Nachfrist setzen
4. Nach Ablauf: Ruecktritt oder Schadenersatzanspruch

## Werkvertrag vs. Dienstvertrag

| Kriterium | Werkvertrag (§ 631 BGB) | Dienstvertrag (§ 611 BGB) |
|---|---|---|
| **Geschuldet** | Ergebnis/Erfolg | Taetigkeit/Bemuehtung |
| **Abnahme** | Ja (Auftraggeber nimmt Werk ab) | Nein |
| **Maengelansprueche** | Ja (Nachbesserung, Minderung) | Nein (kein Erfolg geschuldet) |
| **IT-Beispiel** | Erstellung einer Webanwendung | IT-Beratung, Administrationsservice |
| **Verguetung** | Nach Abnahme | Zeitbasiert/regelmaessig |
| **Kuendigung** | Auftraggeber jederzeit (§ 649 BGB) | Mit Frist |
