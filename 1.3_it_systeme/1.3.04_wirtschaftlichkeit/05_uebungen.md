# Uebungen: 1.3.04 – Wirtschaftlichkeit von IT-Systemen

## Kostenvergleich

**Aufgabe 1:** Ein Unternehmen benoetigt 5 neue Workstations. Vergleiche folgende Optionen ueber 3 Jahre:
- Kauf: 5 × 2.400 EUR
- Leasing: 5 × 80 EUR/Monat
- Welche Option ist guenstiger? Um wie viel?

**Aufgabe 2:** Ein Server kostet 15.000 EUR. Die jaehrlichen Betriebskosten betragen 3.600 EUR. Berechne die TCO ueber 5 Jahre.

# Loesungen

## Aufgabe 1
- Kauf: 5 × 2.400 = 12.000 EUR
- Leasing: 5 × 80 × 36 = 14.400 EUR
- Kauf ist 2.400 EUR guenstiger ueber 3 Jahre.

## Aufgabe 2
- TCO = 15.000 + (5 × 3.600) = 15.000 + 18.000 = **33.000 EUR** ueber 5 Jahre.

## Aufgabe 3
- System A: (0,35×8) + (0,25×6) + (0,20×7) + (0,20×9) = 2,80 + 1,50 + 1,40 + 1,80 = **7,50**
- System B: (0,35×7) + (0,25×9) + (0,20×5) + (0,20×6) = 2,45 + 2,25 + 1,00 + 1,20 = **6,90**
- Empfehlung: **System A** mit 7,50 Punkten.

## Aufgabe 4
- Anbieter A: 6.000 – 15% = 5.100 → – 3% Skonto = 4.947 + 80 Lieferung = **5.027 EUR**
- Anbieter B: 5.800 – 10% = 5.220 + 0 Lieferung = **5.220 EUR**
- Anbieter A ist guenstiger (Differenz: 193 EUR).

## Aufgabe 5
Referenzen/Erfahrung des Anbieters, Servicequalitaet/Reaktionszeit, Nachhaltigkeit/Umweltzertifizierung.

## Aufgabe 6

## Aufgabe 7
- Break-Even: 40.000 / (2.500 – 800) = 40.000 / 1.700 = 23,5 Monate
- **Ab Monat 24** ist On-Premises guenstiger.
